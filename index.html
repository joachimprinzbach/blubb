</ Coding the Website >

Eingesetzte Frameworks

Für das Grid wird Twitter bootstrap (http://getbootstrap.com/) genutzt, für die Icons FontAwesome (http://fortawesome.github.io/Font-Awesome/). Zugriff auf das Backend und Logik mit AngularJS (https://angularjs.org/). Für Scrolleffekte

Backend

Das Backend ist Wordpress und läuft unter der Subdomain blog.joachimprinzbach.de. Der Zugriff erfolgt dann per REST-Services, somit kann das Backend jederzeit durch eine Eigenentwicklung ausgetauscht werden. Dafür hatte leider die Zeit vor dem Abflug nicht mehr gereicht - man muss aber das Rad auch nicht immer neu erfinden.

Responsive Design

Ein paar Elemente des reponsive Designs wurden ausprobiert. Mit media queries und dem mobile First sollte die Seite auch auf kleineren Mobile Devices einigermassen benutzbar sein ;-)

Source Control - Git

Der Code ist auf GitHub frei verfügbar unter https://github.com/joachimprinzbach/website-private.

Build & Deployment

Als Taskrunner fiel die Wahl auf Grunt. Damit werden in erster Linie sämtliche Build tasks ausgeführt (minify, uglify, compile less etc.). Das Deployment erfolgt automatisch durch Heroku.
